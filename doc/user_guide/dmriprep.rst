.. dmriprep:

dMRI Pre-Processings
====================

Introduction
------------

To preprocess and ensure the quality of DWI data, we follow the methodology
described in :footcite:t:`sanjari2025prequal`. It uses the
Pre Qual pipeline :footcite:p:`cai2021prequal`, which was developed using the
software packages MRTrix3 :footcite:p:`tournier2019mrtrix`, FSL
:footcite:p:`jenkinson2012fsl`, and ANTs :footcite:p:`tustison2014ants`
software packages.

Description
-----------

**Steps**: First, volumes with a b-value less than 50 were treated as b0
volumes. The diffusion data was denoised with the MP-PCA-based MRtrix3's ``dwidenoise``
function :footcite:p:`cordero2019mppca`. The images
were then intensity-normalized to the first image and concatenated for
further processing. To correct for susceptibility-induced distortions, motion
artifacts, and eddy currents, FSL's ``topup`` and ``eddy``
:footcite:p:`andersson2016eddy` algorithms were used. Finally, the preprocessed
data were fitted with a tensor model using the MRtrix3's ``dwi2tensor`` function,
employing an iterative reweighted least squares estimator
:footcite:p:`veraart2013dti`.

**Quality control**: Results were assessed evaluated both qualitatively,
to identify gross errors, and quantitatively through a structured
three-step approach. First, the preprocessed data were assessed following
the methodology described by :footcite:t:`lauzon2013qa`. A brain parenchyma
mask - excluding cerebrospinal fluid - was generated by eroding the brain
mask derived from the average b0 image using FSL's ``bet2`` function
:footcite:p:`smith2002bet2`.Tensor fits within the masked region were then
backpropagated through the diffusion model to reconstruct the original signal,
and model fit quality was quantified using a modified pixel-wise chi-squared
metric computed per slice and per volume.
In the second step, DTI-derived fractional anisotropy (FA) image were
non-rigidly registered to the ICBM FA MNI atlas using ANTs. This atlas
includes 48 white matter bundles. The mean FA for each bundle were extracted
and evaluated for physiologic congruence.
Finally, gradient orientations were inspected using MRtrix's ``dwigradcheck``
:footcite:p:`jeurissen2014grad`.
We automate this process by checking the mean FA values of five specific
bundles:
    
- Genu_of_corpus_callosum_med 
- Body_of_corpus_callosum_med
- Splenium_of_corpus_callosum_med
- Corticospinal_tract_L_med
- Corticospinal_tract_R_med
    
Mean FA values must range in [0.3, 0.75].
